### 4. Modul Manajemen Inti (Fungsionalitas *Troubleshooting*)

Adaptasi elemen *To-do List* dan *Transaction Log* menjadi pusat kontrol:

#### A. Panel Manajemen Masalah (*To-do List* Adaptasi)

* **Judul:** **Aksi Wajib & Peringatan Sistem**
* **Isi:** Daftar prioritas tugas yang membutuhkan intervensi Admin:
    1.  **Lowongan Baru Menunggu Review (Hasil AI):** Link ke *AggregatedJobQueue*.
    2.  **Laporan Penipuan Baru:** Link ke *list* laporan yang diajukan Kandidat.
    3.  **Permintaan *Refund* Tertunda:** Link ke *queue* transaksi yang menunggu persetujuan *refund*.
    4.  **Verifikasi Perusahaan Tertunda:** Link ke dokumen legalitas yang perlu dicek.

#### B. Tabel Utama (*Transaction & Activity Log*)

* **Tabel 1: Transaksi Keuangan** (Adaptasi dari *Transactions* di referensi)
    * **Kolom Wajib:** Tanggal, ID Transaksi, Tipe (Booster, Slot), Jumlah (Rp), **Status**.
    * **Fungsi:** *Search bar* di atas tabel untuk mencari berdasarkan ID atau jumlah. Tombol *Export CSV* (warna **Lime Green**) untuk laporan keuangan bulanan.

* **Tabel 2: Log Sistem Kritis (*Activity Log* Adaptasi)**
    * **Kolom Wajib:** Waktu, Pengguna (ID Admin/Sistem), Aksi, Status.
    * **Fungsi:** Mencatat semua *event* penting: Server *restart*, API *error* (500), Perekrut diblokir, Admin menyetujui lowongan AI.
