VERIFIED BADGE IMPLEMENTATION
═══════════════════════════════════════════════════════════

### Logic Check

**Query saat render job card:**
```
SELECT 
  jobs.*,
  employers.company_name,
  employers.company_logo_url,
  employers.subscription_plan,
  employers.has_verified_badge
FROM jobs
JOIN employers ON jobs.employer_id = employers.id
WHERE jobs.status = 'published'
```

**Render Logic:**
```
FOR each job:
  Display job title
  Display location
  
  Display company logo (real or fallback)
  Display company name
  
  IF employer.subscription_plan IN ['starter', 'professional', 'enterprise']:
    Display verified badge ✓ (blue)
  ENDIF
  
  Display meta info
  Display badges (featured, urgent) if applicable
  Display button
```

### Verified Badge Component

```
<div class="company-info">
  <img src="logo.jpg" class="company-logo" />
  <span class="company-name">Enterprise Test Company</span>
  
  <!-- Verified Badge (Conditional) -->
  <?php if ($employer->subscription_plan !== 'free'): ?>
    <span class="verified-badge">
      <svg class="verified-icon">
        <circle fill="#1DA1F2" />
        <path fill="#FFFFFF" d="M checkmark path" />
      </svg>
    </span>
  <?php endif; ?>
</div>
```

**CSS:**
```

  display: inline-flex;
  align-items: center;
  margin-left: 6px;
}

.verified-icon {
  width: 18px;
  height: 18px;
  fill: #1DA1F2;
}
```
