═══════════════════════════════════════════════════════════
MASALAH 3: BUTTON "LAMAR CEPAT" LANGSUNG SUBMIT TANPA FORM
═══════════════════════════════════════════════════════════

**Problem:**
- Klik "Lamar Cepat" → Langsung submit aplikasi
- Tidak ada konfirmasi
- Tidak ada review profile
- User tidak bisa review sebelum submit

**Fix: MULTI-STEP MODAL FORM**

### Backend

**POST /api/jobs/:jobId/quick-apply**
Purpose: Initialize apply flow (check can apply)
Response:
- can_apply: boolean
- user_profile: name, email, phone, resume
- employer_questions: array (if any)

**POST /api/jobs/:jobId/apply**
Purpose: Final submit (setelah user konfirmasi)
Actions:
- Insert to job_applications table
- Send email to employer
- Send confirmation email to job seeker
- Increment applications_count

### Frontend: Multi-Step Modal

**STEP 0: Click "Lamar Cepat"**
→ Call POST /api/jobs/:jobId/quick-apply
→ Show loading
→ If can_apply: Open Step 1 modal
→ If cannot: Show error "Anda sudah melamar"

**STEP 1: Review Profile**

Modal content:
```
┌────────────────────────────────────┐
│ Lamar ke [Job Title]               │
├────────────────────────────────────┤
│ Review Profil Anda                 │
│                                    │
│ ✓ Nama: [API data]                │
│ ✓ Email: [API data]               │
│ ✓ Telepon: [API data]             │
│ ✓ Resume: [filename]               │
│                                    │
│ Apakah informasi sudah benar?      │
│                                    │
│ [Edit Profil] [Lanjutkan →]       │
└────────────────────────────────────┘
```

**STEP 2: Cover Letter (Optional)**

```
┌────────────────────────────────────┐
│ Surat Lamaran (opsional)           │
├────────────────────────────────────┤
│                                    │
│ [Textarea]                         │
│ Tulis surat lamaran... (max 500)   │
│                                    │
│ 0/500 karakter                     │
│                                    │
│ [← Kembali] [Lanjutkan →]         │
└────────────────────────────────────┘
```

**STEP 3: Final Confirmation**

```
┌────────────────────────────────────┐
│ Konfirmasi Lamaran                 │
├────────────────────────────────────┤
│                                    │
│ Anda akan melamar ke:              │
│ -  [Job Title]                      │
│ -  [Company Name]                   │
│                                    │
│ Apakah Anda yakin?                 │
│                                    │
│ [Batal] [Ya, Kirim Lamaran]       │
└────────────────────────────────────┘
```

Click "Ya, Kirim":
→ Call POST /api/jobs/:jobId/apply
→ Show loading
→ On success: Step 4

**STEP 4: Success Screen**

```
┌────────────────────────────────────┐
│          ✅                        │
│  Lamaran Berhasil Dikirim!        │
│                                    │
│ Lamaran Anda telah dikirim ke      │
│ [Company Name]                     │
│                                    │
│ Lacak status lamaran di:           │
│ Lamaran Saya                       │
│                                    │
│ [Lihat Lamaran Saya] [Tutup]      │
└────────────────────────────────────┘
```

Success screen styling:
- Background: #f0fff4 (hijau muda)
- Icon: ✅ 64px
- Buttons: Lime primary

### JANGAN:
- Langsung submit tanpa konfirmasi
- Skip form review
- Auto-apply tanpa user action
